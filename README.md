Task Manager App

Description:
A simple Android application to create, view, delete, and mark tasks as completed.
Data is stored locally using Room database.

Features:
- Add new tasks
- View task list
- Delete tasks
- Mark tasks as completed
- Data persists after restart

Technologies:
- Kotlin
- Room Database
- RecyclerView
- ViewModel

Secure Coding:
1. Input validation prevents empty or invalid task entries.
2. Local database storage avoids insecure file storage.
